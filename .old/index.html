Password Service

The password service is my most recent project that was built during my internship at Windstream. It is a website for employees to perform CRUD operations via API calls on sensitive passwords for databases or any other type of connection. It features a sleek UI, complete with swagger documentation, along with many other features, such as real-time password encryption and decryption and dex authentication. The back-end is built in Node.js and Express.js, while the front-end is built in typical HTML/CSS/JS and Bootstrap. MongoDB and Docker were also utilized.

Sorting Visualizer

The sorting visualizer is a desktop application that allows the user to populate an array with random numbers that adhere to user-defined bounds. Then, the user can select their desired sorting method and sort the generated array. Each sorting method will finally be visualized via the bars. The sorting visualizer is built entirely within Python3, notably utilizing its GUI library, TkInter.

URL Shortener

The URL Shortener is a very simple service that mimics bit.ly. Users can input URLs to generate a smaller, concise URL that is distinct from any other, and it still leads to the original URL. This process functions via an API. When the user inputs a foreign URL, it is appended to the database, along with a distinct key that maps to the original URL. If the user implements an existing URL, the database will pull it without generating a new URL. The back-end was built with Python's Flask, while the front-end is built with typical HTML/CSS/JS and AJAX. The database is MongoDB.

Coffee Kiosk

The coffee kiosk was one of my earlier projects. The program mimics Panera's kiosk system, although on a much smaller scale. A user can select an item and append it to the cart, along with any add-ons they so choose. Finally, the user can "checkout", which is just a small message box that displays their total and clears the cart. The coffee kiosk is built solely in C#'s Window's Forms, and it utilizes a number of frameworks, such as BunifuUI and MetroUI.
