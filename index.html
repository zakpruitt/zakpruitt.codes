Data Structure Visualizer  

This desktop application visualizes Linked Lists, Stacks, Queues, Binary Search Trees, and Heaps using C#'s Windows Forms and Microsoft Automatic Graph Layout (MSAGL). It contains method information (parameter information), time complexity, an analysis of each method, and a node color legend. The applications has many tools, such as a modular graph and the ability to save, print, and load extant graphs.

Sorting Visualizer  

The sorting visualizer is a desktop application that allows the user to populate an array with ran-dom numbers that adhere to user-defined bounds. Then, the user can select their desired sorting method and sort the generated array. Each sorting method will finally be visualized via the bars.
The sorting visualizer is built entirely within Python3, notably utilizing its GUI library, TkInter.

URL Shortener  

The URL Shortener is a very simple service that mimics bit.ly. Users can input URLs to generate a smaller, concise URL that is distinct from any other, and it still leads to the original URL. This process functions via an API. When the user inputs a foreign URL, it is appended to the database, along with a distinct key that maps to the original URL. If the user implements an existing URL, the database will pull it without generating a new URL. The back-end was built with Python's Flask, while the front-end is built with typical HTML/CSS/JS and AJAX. The database is MongoDB.
